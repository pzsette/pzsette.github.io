Cultmedia Web Annotator

Barchelor Thesis project: A web application for emotional video-compositing

The goal of this thesis work was the realization of a web-application that allows to manage and compose together a set videoclips trough a specifical video-editor, suggesting and helping the user during the choice of files relyng of their emotional characterization. The main user interfaces are two:

  • Video-editor, which allows to merge video sequences taken from the databes and filtering them trough specific paramenters.
  • Annotator, which allow to a group of different users to mark every video clip with a praticular emtional value.

Tools

The main tools used are:

  • Django.
  • Docker.
